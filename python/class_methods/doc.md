# Функции

### Написание функции
```
# Все сопоставление аргументов по позиции или по имени
    def funс(name)
# Аргументу назначено значение по умолчанию
    def funс(name=value)
# Объединяет все дополнительные аргумены в кортеж (передача в функцию производится нескольких значений)
    def funс(*name)
# Находит и объединяем все дополнительные аргумены в словарь (когда функции передаю самые разные значения и их нужно вынимать
    def funс(**name)
# Именные аргументы отбираются отдельно а остальные уходят в кортеж
    def funс(*args, name) 
    def func(*, name=value)
```

### Вызовы функции
```
# Сопоставления роизводятся по позиции аргумента
    func(value) 
# Именнованный аругмент. Сопоставления роизводятся по указанному имени
    func(name=value) 
# Все объекты в указанной последовательности передаются как отдельные позиционные аргументы
    func(*sequence) 
# Все пары ключ / значение в указанном словаре передаются как именованные аргументы
    func(••dict)
```
